Installing Eclipse EGit Plugin

Context

If you are using Git as the source control system, this plugin needs to be installed to access all the EGit features and perform actions on Git.

Procedure

  1. From the main menu, choose  Windows  Preferences  Install/Updates  Available Software Sites .
  2. In the Available Software Site page, check if Eclipse EGit update site (http://download.eclipse.org/egit/updates-1.3/ ) is available and enabled.
  3. If the update site is available but not yet enabled, select the check box next to the update site name.
  4. If the update site is not available, follow the steps below:
    1. Choose Add... in the Available Software Sites page and enter a name for the update site and the location as http://download.eclipse.org/egit/updates-1.3/ .
    2. Choose OK.
  5. In the Preferences dialog, choose OK.
  6. from the main menu choose  Help  Install New Software… .
  7. In the Install wizard, select the EGit update site from the Work with: dropdown list.
  8. Expand the Eclipse Git Team Provider plugin and select Eclipse EGit feature and choose Next.
  9. In the Install Details page, choose Next.
  10. In the Review Licenses page, accept the terms of licensing agreement and choose Finish.
  11. In the Software Updates dialog, choose Restart Now.
