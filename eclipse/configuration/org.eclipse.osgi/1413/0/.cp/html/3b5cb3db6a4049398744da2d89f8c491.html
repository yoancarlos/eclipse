Accessing Tooling Features Available in Internal Mode

Context

You use this task to access the tooling features that are not released but are made available to you in the internal mode for testing or validation.

Note The features that are available internally will be labeled as (Available in Internal Mode) in the Developer's Guide document.

Procedure

  1. In the folder containing Eclipse application, open the eclipse.ini file using a notepad.
  2. In the eclipse.ini file, enter the property to enable internal mode as -Dcom.sap.it.internal=true.
  3. Restart the Eclipse application.
