Editing Integration Flow Configurations

Context

You use this procedure to edit endpoint and certificate configurations for a deployed integration flow. The application displays configuration details only for externalized parameters.
Note To externalize parameters for a deployed integration flow, access the Eclipse environment of SAP HANA Cloud Integration.

Procedure

  1. To edit the endpoint and certificate configurations of the required integration flow bundle, go to My Projects page and choose the icon (Configure)
    Note You can also access this icon from two other places:
    • Within the graphical representation of the integration flow.
    • From the top right corner of the My Projects page.
  2. To change the endpoints, enter the details of the new endpoint in the Endpoints tab page.
  3. To change certificate details or add new certificates, select the Certificates tab and make the required changes.
  4. To deploy the changes on the current tenant, choose Deploy.
  5. To view the changes, choose the icon (Refresh) on the My Projects page.
