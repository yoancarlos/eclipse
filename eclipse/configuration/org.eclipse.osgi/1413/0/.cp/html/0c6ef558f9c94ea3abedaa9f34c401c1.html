Defining EDI Extractor

EDI Extractor enables you to extract EDI headers and transfer to camel headers. This element extracts data from the incoming EDI document and adds it to the exchange such that this information can be used further in message processing. EDI extractor can read both flat file and XML format.

Context

Procedure

  1. Open the integration flow in the Model Configuration editor.
  2. Choose  Add Message Transformers  EDI Extractor  from the context menu of a connection within the pool.
  3. On the EDI Extractor properties tab page, define the parameter.
  4. Save the changes.

Example

The element creates camel headers and populates it with respective extracted values. For example, if following segments are part of message payload then respective headers and values for the same are given in the table below.
