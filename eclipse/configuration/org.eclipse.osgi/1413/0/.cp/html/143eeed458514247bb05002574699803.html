Assigning a Local Integration Process

Prerequisites

You have created a local integration process.

Context

Procedure

  1. In the palette, choose  Tasks  Process Call  and drop the shape into the modelling area.
  2. Position the cursor on the Process Call shape and in the context menu choose Assign Local Integration Process.
  3. Enter the name of the local integration process you like to assign.
  4. Choose Ok.
