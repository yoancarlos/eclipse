Editing Scripts of a Mapping

You use this procedure to modify scripts of a mapping.

Context

Procedure

  1. Launch SAP HANA Cloud Integration Web application.
  2. Choose you required integration package in the Overview section of the Design tab.
  3. In the Artifacts section of the <integration package name> editor, choose the artifact.
  4. In the <artifact name> editor, choose the required mapping.
  5. Choose Edit.
  6. In the Mapping section, choose the mapping name to open the mapping editor.
  7. Choose the function whose script you want to change.
  8. In the Mapping expression section, double-click on the function to edit it.
  9. Choose Save.
