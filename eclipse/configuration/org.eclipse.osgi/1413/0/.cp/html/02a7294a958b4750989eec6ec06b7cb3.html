Export MPL

Context

You enable this element to export and analyze a trace.

Procedure

Select a processed message in Message Monitoring editor. On clicking the same, Message Processing Log dialog box opens. Select one of the following from the dialog box:
  • Export Log – Exports message flow sequence only
  • Export Log with Trace – Exports message flow sequence, message payload and headers at different trace points
  • Export Log with Trace and Integration Project
Note

  • You can export only message processing log for a configure-only integration flow.

  • You cannot export MPL for multiple messages. Please select one message only.
