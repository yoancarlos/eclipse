Working with Product Profiles

Product profile describes the capabilities supported by products such as Success Factor Adapter, Splitter and so on. You can consume these capabilities at the time of designing integration flows.

Acces product profiles in HCI

  • Access default Native profile that is bundled with eclipse.
  • Connect to any cluster to access the profile of the server.
  • Import the profiles in .zip format from your local file system by choosing Import in the Product Profiles dialog.

Export product profiles

Choose Exportin the Product Profiles dialog, to export the profiles in .zip format.

Assign product profile to integration project

  • Project level assignment

    1. Open an integration project, go to Project Configuration -> Properties tab.
    2. From the Product Profile drop down list, select the required profile.
  • Workspace level assignment
    1. In Eclipse, open Windows -> Preferences-> SAP HANA Cloud Integration -> Product Profiles.
    2. Select the required profile.

If you have selected the option as Native at the project level configuration, then the workspace level product profile settings overrides the project level. The system then executes checks at the workspace level. If you have selected any other profile at project level, then the system executes all the checks against that profile.
