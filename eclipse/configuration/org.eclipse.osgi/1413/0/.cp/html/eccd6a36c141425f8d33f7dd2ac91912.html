Viewing Integration Packages

Context

You use this procedure to manage different types of Integration Packages. These integration packages contain various sets of artifacts that an integration expert requires for modeling, and customizing integration scenarios. You can download artifacts of an integration package only if you are a licensed user of the application. As a prospective customer, you only have viewing rights.

Procedure

  1. Launch SAP HCI Spaces by accessing the URL provided by SAP.
  2. If you are a new user, choose Signup.
  3. If you have user credentials, choose Login.
    Note For a prospective customer, use the following credentials to log on to the application:
    • Login: hcidemo
    • Password: Secure22
  4. Select the Catalog tab. The list of integration packages appears. By default, the application sorts them in ascending order.
  5. Select the required integration package.
  6. If you want to view the details of an integration package, choose <Integration Package name>.
    Note The integration package contains five types of artifacts. They are:
    • Data Integartion
    • Integration Flows
    • Files
    • URL
    • Value Mappings
  7. To download an artifact, choose (Click to download).
  8. To download all the artifacts, choose Download All. You can only download integration flow, files, and value mappings. Once downloaded, your local file system stores the artifact with the name as <IntegrationPackage>_artifact.zip file. To use the artifact in eclipse, you need to import it in eclipse after extracting it from the zip file.
