Add an Integration Flow to an Integration Package

You can add an integration flow to an integration package.

Prerequisites

  • You have logged-on to the Web application with the integration developer role.
  • You have created an integration package.

Context

An integration flow is a graphical representation of the flow and processing of messages between two or more participants using an integration runtime platform, ensuring successful communication.

You can execute the following steps to create a new integration flow or upload an existing one.

Procedure

  1. Choose    Design .
  2. Choose the integration package in which you want to add an integration flow.
  3. Choose Edit.
  4. Choose Add.
  5. Select Process Integration.
  6. In Add Process Integration Artifact dialog, choose create or upload option as per the requirement.
  7. if you want to create a new integration flow, execute the following substeps.
    1. Choose Create.
    2. Enter integration flow details.

      You can choose product profile for the integration flow from Product Profile field. The integration flow templates used during creation adheres to the latest version of a component available in product profile.

    3. Choose OK.
  8. If you want to upload an integration flow, execute the following substeps.
    1. Choose Upload.
    2. Choose Browse to select the integration project archive saved in your local file system.
    3. Choose OK.
      Note

      For create or upload the name of integration flow should begin with alphabets 'Aa' to 'Zz' or underscore '_' . It can also contain numbers '0-9', space ' ', dot '.' and hyphen '-'.

      You can access the integration flow that you created or uploaded in the Artifacts table.
