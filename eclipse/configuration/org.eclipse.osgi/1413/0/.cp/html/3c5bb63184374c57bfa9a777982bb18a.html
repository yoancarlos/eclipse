Decrypting the Message Content

Context

You perform this task to decrypt messages received from a participant on the cloud.

Note that the related keystore must contain the private key, otherwise, decrypting the message content will not work.

Procedure

  1. Open the <integration flow>.iflw in the Model Configuration editor.
  2. Right-click on a connection within the pool and choose Add Security Element  Content Decryptor .
  3. In the Model Configuration editor, select Decryptor.
  4. In the Properties view, enter the details to decrypt the content of incoming message.
  5. Save the changes.
