Working with PI Tools

 

You use this procedure to enable you in achieving communication between two or more heterogeneous applications or systems that exist within or across business landscapes. You can have the role of a designer or configurator at different stages of taskflow to perform activities on objects involved in process integration. For each role, PI Tools offers dedicated perspectives and interfaces to interact with the objects in ES Repository and Integration Directory under one platform.

Prerequisites

You have downloaded the latest version of SAP NetWeaver Developer Studio from SAP Service Marketplace.

For information, follow the instructions in the Netweaver Developer Studio Installation Guide.

Procedure

  1. Learn about integration flow and get familiar with the new features offered in the user interface of this tool.

    More information: Understanding Integration Flows, Understanding User Interface.

  2. Establish a connection with the Integration Directory and the ES Repository.

    More information: Connecting to the Integration Directory

  3. Perform all the activities of a Configurator such as configuring, deploying and monitoring the PI objects.

    More information:Working with Integration Flows

  4. Perform all the activities of an Administrator such as maintaining the PI objects and keeping a check on the authorizations.

    More information: Administering PI Objects
